Aplicación Web Progresiva

En 2015, el diseñador Frances Berriman y el ingeniero de Google Chrome Alex Russell acuñaron el término "aplicaciones web progresivas" para describir las aplicaciones que aprovechan las nuevas funciones compatibles con los navegadores modernos, incluidos los service worker web y los manifiestos de aplicaciones web, permitiendo actualizar las aplicaciones web a aplicaciones web progresivas en su sistema operativo (SO) nativo.
