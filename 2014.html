Aplicación web de una sola página

Es un modelo de aplicación creado como una solución al modelo de construcción de softwareconvencional, denominado MPA (Multiple Page Application).
La solución que SPA soluciona es no redireccionar a los usuarios a una nueva página, consumiendo menos tiempo y datos.
Vue.js es un framework de JavaScript de código abierto destinado para diseñar SPA. Fue creado por Evan You y lanzado por primera vez en febrero de 2014
